Constructs a FlatMatrix from given dimensions and a flat data vector.

Creates a matrix with r rows and c columns, initialized from a flat vector m stored in row-major order.

Parameters
rNumber of rows (must be > 0).
cNumber of columns (must be > 0).
mFlat vector containing matrix elements in row-major order.
Exceptions
std::invalid_argumentif r == 0 or c == 0.
std::invalid_argumentif m.size() != r * c.
Note
The vector m is moved into the internal storage (no copy is made).

template<typename T >
FlatMatrix< T > LinearAlgebra::FlatMatrix< T >::operator~ ( ) const

Returns a transposed copy of the matrix.

Implements matrix transposition operator (~). For a matrix A of size r x c, returns a new matrix At of size c x r such that At[i][j] = A[j][i].

A = [[1,2,3], At = [[1,4,7],
[4,5,6], => [2,5,8],
[7,8,9]] [3,6,9]]
Returns
Transposed matrix (original matrix is not modified).
