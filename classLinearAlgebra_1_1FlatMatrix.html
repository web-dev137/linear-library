FlatMatrix< T > LinearAlgebra::FlatMatrix< T >::operator~ ( ) const

Returns a transposed copy of the matrix.

Implements matrix transposition operator (~). For a matrix A of size r x c, returns a new matrix At of size c x r such that At[i][j] = A[j][i].

A = [[1,2,3], At = [[1,4,7],
[4,5,6], => [2,5,8],
[7,8,9]] [3,6,9]]
Returns
Transposed matrix (original matrix is not modified).
