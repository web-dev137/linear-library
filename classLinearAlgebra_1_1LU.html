Detailed Description

template<typename T, typename MatrixType>
class LinearAlgebra::LU< T, MatrixType >

LU decomposition with partial pivoting in-place.

Decomposes the matrix \(A\) such that

\[ P A = L U \]

where \(P\) is a permutation matrix, \(L\) is a unit lower triangular matrix, and \(U\) is an upper triangular matrix.

The decomposition is stored in-place.
